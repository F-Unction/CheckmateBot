# CheckmateBot
CheckmateBot

框架：<https://github.com/Polythefake/KanaBot>

基本版：<https://gitee.com/OI-Master/checkmatebot/>

**特别提示：验证码无需用户手动输入**

验证码破解器使用了[腾讯云OCR](https://cloud.tencent.com/document/product/866)

在使用前，请在目录下创建`config.json`，填写以下内容：
```json
{
  "username": "xxx",
  "password": "xxx",
  "roomID": "xxx",
  "secretId": "xxx",
  "secretKey": "xxx"
}
```
